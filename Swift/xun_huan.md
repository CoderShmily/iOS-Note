# 循环

## OC风格的 `for`

```swift
// 传统写法
for var i = 0; i < 10; i++ {
    print(i)
}
```

## Swift风格的 `for`

```swift
// 遍历 0 ~ <10
for i in 0..<10 {
    print(i)
}

print("---")

// 遍历 0 ~ 10
for i in 0...10 {
    print(i)
}
```

* 阶段性小结
    * `Swift` 中使用 `in` 关键字标示循环的范围
    * `0..<10` 表示从0到9
    * `0...10` 表示从0到10
    * 注意之间不能出现空格

## 特殊写法

```swift
for _ in 0...10 {
    print("hello")
}
```

* 阶段性小结
    * 如果不关心循环本身的索引，可以使用 `_` 忽略
    * 这一技巧在之前的分支演练中出现过
